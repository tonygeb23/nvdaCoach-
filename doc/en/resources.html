NVDA Coach — Additional Training and Help

NVDA Coach Resources

  • NVDA Coach User Guide — Full documentation for the NVDA Coach add-on, including all keyboard shortcuts, lesson descriptions, and customization options.

Web Accessibility and Screen Reader Resources

  • WebAIM — Practical guides and articles on web accessibility, including screen reader techniques and keyboard navigation.
  • WebAIM: Using NVDA to Evaluate Web Accessibility — A practical guide to using NVDA for web navigation and accessibility testing.
  • W3C Web Accessibility Initiative (WAI) — International standards and guidelines for accessible web design.
